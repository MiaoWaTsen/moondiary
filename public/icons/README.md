# PWA Icons

Add the following icon files for PWA support:

- `icon-192x192.png` - 192x192 pixels
- `icon-512x512.png` - 512x512 pixels

These icons should:
- Be square PNG images
- Have a transparent or solid background
- Use the Moodiary logo or app icon
- Support maskable icon format (safe area in center)

You can use tools like:
- [PWA Asset Generator](https://github.com/nickmessing/pwa-asset-generator)
- [Favicon.io](https://favicon.io/)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
